git init - initialize the git repository

git status - it checks the status or it checks the committed files

git add index.js - it add index.js alone

git add src - it add all of the folders including the sub folders

git add . (or) git add -A => it adds all the changes 

git commit -m => it commits all the data [-m => message]

#config
git config --global user.email 'xyz@gmail.com'

git config --global user.name 'xyz'

git commit -m "Test" --amend => Means it commits to the previous version

git log => It shows all the versions

git reset && git checkout -- => rseset or discards the changes
